# v0.0.2 Fix - Kategori Ayrı Sayfa Hazırlığı

- Kategori bağlantıları aynı sayfada açılmayacak şekilde düzeltildi.
- `/categories` kategori liste sayfası hazırlandı.
- `/categories/korku` kategori detay sayfası hazırlandı.
- `/categories/aksiyon` kategori detay sayfası hazırlandı.

- Supabase, YouTube API, RAWG, Steam ve admin paneli eklenmedi.
