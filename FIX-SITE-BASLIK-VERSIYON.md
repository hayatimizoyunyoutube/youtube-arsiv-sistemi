# Site Başlığı Versiyon Fix

Bu paket yeni sürüm değildir. v1.3.1 üzerinde kalan eski sekme/site başlığı sürüm yazısı düzeltildi.

## Düzeltildi

- `index.html` tarayıcı sekmesi başlığı `v1.2.5` yerine `v1.3.1` yapıldı.
- Site adı ve sekme/title tarafı mevcut sürümle eşitlendi.
- Supabase SQL gerekmez.
- Yeni `.env` gerekmez.
- Veri/yetki sıfırlama yok.
