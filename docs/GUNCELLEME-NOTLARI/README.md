# Güncelleme Notları

Bu klasörde her sürüm için ne yapıldığı, neyin eklenmediği ve bir sonraki adıma ne zaman geçileceği yazılır.

Kural:
- Sürüm atlamadan ilerlenir.
- Her sürüm bittikten sonra notu buraya eklenir.
- `.env`, Supabase ve API anahtarları sadece gerekli sürüm geldiğinde eklenir.
- Eski bozuk paketlerden dosya taşınmaz.
