# v0.0.1 - BAT Dosyaları Eklendi

## Eklenenler

- `01-projeyi-temizle-git-koru.bat`
  - Geçici dosyaları temizler.
  - `.git` klasörünü silmez.
  - `docs`, `README.md`, kaynak dosyalar korunur.

- `02-githuba-gonder.bat`
  - Git yoksa hata mesajı verir.
  - `.git` yoksa `git init` yapar.
  - Branch adını `main` yapar.
  - GitHub repo adresini sorar.
  - Commit ve push işlemini yapar.

## Not

Bu sürüm hâlâ temiz başlangıç sürümüdür. Supabase, API, `.env`, YouTube, Steam ve RAWG entegrasyonları daha sonraki sürümlerde sırayla eklenecektir.
